# A Jupyter Kernel that works with the latest version of Mathics.
This is a Wolfram Mathematica alternative.

## Status
- Version: 0.0.1 Beta.

## Working.
- Symbolic calculus: Derivative, Integrate, Solve, Solve, D,...
- Number theory.
- Graphics; Plot 2d functions.
